
## Minha Escala
 
O Minha Escala é uma aplicação criada para atender o controle de escala de voluntário de uma igreja.
 
 
## Tecnologias 
 
Aqui estão as tecnologias utilizadas nesse projeto.
 
* Reactjs
* Nextjs
* Styled Components
* MSQL database
* Docker compose
